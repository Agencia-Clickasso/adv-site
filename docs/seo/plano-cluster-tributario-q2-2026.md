# Plano de Cluster Tributário - Q2 2026

## Objetivo

Transformar o site em um ativo orgânico com foco principal em Direito Tributário para empresas e profissionais, conectando:

- páginas comerciais;
- artigos BOFU/MOFU;
- CTA para contato;
- autoridade local em São Bernardo do Campo.

## Página pilar

- Página principal: `/areas/direito-tributario`
- Papel: concentrar intenção comercial, autoridade temática e conversão
- CTA principal: `Solicitar Atendimento`

- Intenção: avaliação / comercial

- Página de apoio comercial: `/areas/direito-tributario`

### Cluster 5: Autoridade local

- Intenção: local / comercial
- Objetivo: consolidar relevância geográfica
- Página de apoio comercial: `/areas/direito-tributario`

## Regras de interlink

- Todo post do cluster deve linkar para `/areas/direito-tributario`
- Todo post BOFU deve ter link para `/#contact`
- Todo post MOFU deve apontar para pelo menos 2 posts BOFU
- A home deve destacar os 3 posts BOFU prioritários
- O hub `/blog` deve manter tributário como primeira categoria visível
