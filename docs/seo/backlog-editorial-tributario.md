# Backlog Editorial SEO - Direito Tributário

## Objetivo

Ampliar a cobertura orgânica qualificada com foco em Direito Tributário para empresas e profissionais, priorizando buscas com intenção comercial e consultiva.

## Prioridades

- `P1` = alta intenção comercial (BOFU)
- `P2` = intenção de avaliação/educação com impacto comercial (MOFU)
- `P3` = topo de funil / autoridade temática

## Template de Artigo SEO (padrão)

### 1. Metadata

- `title`: palavra-chave principal + benefício + marca
- `description`: 140-160 caracteres com serviço, benefício e CTA implícita
- `canonical`: `/blog/<slug>`

Exemplo:

- `title`: `Planejamento Tributário para Empresas: Como Reduzir Riscos | Lucimeire Xavier Advocacia`
- `description`: `Entenda como o planejamento tributário ajuda empresas a reduzir riscos fiscais e organizar a carga tributária com segurança jurídica.`

### 2. Estrutura de Conteúdo

- `H1`: igual ao foco da pauta (sem clickbait)
- Introdução: problema + impacto + promessa do artigo
- `H2`: cenário/risco
- `H2`: fundamentos legais/práticos
- `H2`: passos práticos / checklist
- `H2`: quando procurar assessoria especializada
- Conclusão: resumo + CTA

### 3. CTA Padrão (rodapé do artigo)

- Convite para consulta tributária
- Link para `/areas/direito-tributario`
- Link para `/#contact`
- Reforço de atendimento em São Bernardo do Campo e online (se aplicável)

## Mapa de Linkagem Interna (SEO)

### Fluxo principal

- `Home (/)` -> `Área Tributária (/areas/direito-tributario)` -> `Posts tributários` -> `Contato (/#contact)`

### Regras de interlink

- Todo post tributário deve linkar para `/areas/direito-tributario`
- Toda página de serviço tributário deve linkar para `/#contact`
- Blog index deve destacar categoria tributária nas primeiras posições
- Posts de topo de funil (`P3`) devem apontar para posts `P1` e para a página de serviço
